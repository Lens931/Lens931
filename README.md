## 🚀 Projets principaux

### 🥃 Omerta Empire

Jeu navigateur mafia/prohibition développé en solo, inspiré des anciens jeux de stratégie web comme Vendetta, OGame et les univers criminels des années 1930.

**Stack :** React, TypeScript, FastAPI, PostgreSQL, Redis, Docker, Apache, WebSocket.

**Fonctionnalités :**

- économie persistante ;
- bâtiments et salles améliorables ;
- files d’attente de construction ;
- PvP, espionnage, transports et rapports de combat ;
- interface Art Déco mafia ;
- panel admin ;
- système temps réel via WebSocket.

---

### 🛡️ LSVAC Anti-Cheat

Système de protection pour serveurs FiveM/QBCore, conçu pour améliorer la sécurité, détecter les comportements suspects et protéger les ressources critiques.

**Fonctionnalités :**

- détection côté client et serveur ;
- logs Discord détaillés ;
- système de ban persistant ;
- vérification d’intégrité des fichiers ;
- protection contre certains abus d’events ;
- surveillance des entités suspectes.

---

### 🕷️ RedTigerTrap / Obsidian Firewall

Honeypot Flask conçu pour détecter les bots, scanners web, tentatives d’intrusion et comportements suspects.

**Fonctionnalités :**

- endpoints leurres ;
- scoring IP ;
- logs enrichis ;
- alertes Discord ;
- intégration Fail2Ban ;
- détection de patterns suspects ;
- ralentissement volontaire des bots.

---
